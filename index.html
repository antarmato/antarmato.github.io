Element/Property Description Example/Usage

p Defines a paragraph in HTML.

a Creates hyperlinks to navigate to other pages or sections. Visit
img Embeds images in a webpage.
ul and ol Creates unordered and ordered lists.
  • Item 1
div Defines a container for grouping elements.
video Embeds a video in a webpage.
color Sets the text color. color: blue;
font-size Sets the font size of text. font-size: 16px;
padding Adds space inside an element around its content. padding: 10px;
margin Adds space outside an element, separating it from others. margin: 10px auto;
display Specifies how an element is displayed. display: inline-block;
background-image Sets a background image for an element. background-image: url("image.jpg");
position Positions an element (static, relative, absolute, fixed). position: absolute; top: 10px;
box-sizing Defines how the total size of an element is calculated. box-sizing: border-box;
opacity Sets the transparency level of an element. opacity: 0.5;
